## Opis projekta
Ovaj projekat predstavlja CRUD web aplikaciju za upravljanje korisnicima.
Aplikacija je razvijena koristeći:
- MySQL (baza podataka)
- PHP (backend)
- HTML, CSS i Bootstrap (frontend)
- JavaScript (validacija i dodatne funkcionalnosti)

Funkcionalnosti aplikacije:
- Dodavanje novog korisnika (Create)
- Prikaz svih korisnika (Read)
- Izmjena podataka korisnika (Update)
- Brisanje korisnika (Delete)

Bonus funkcionalnosti:
- Validacija forme u JavaScriptu
- SweetAlert potvrda brisanja
- Pretraga korisnika
- Paginacija
